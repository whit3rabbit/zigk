<objective>
Implement splice, tee, vmsplice, and copy_file_range syscalls for kernel-side data transfer between file descriptors and pipes.

Purpose: Enable data movement between FDs and pipes without user-space buffer round-trips. Since zk has no page cache, these use kernel buffer copies (same pragmatic approach as the existing sendfile implementation) rather than true zero-copy page remapping.

Output: 4 new syscalls working on both x86_64 and aarch64, with userspace wrappers and integration tests.
</objective>

**CRITICAL: Pipe detection pattern.** To determine if an FD is a pipe, compare its ops pointer against the exported `pipe_ops` from `pipe.zig`. The pipe module currently declares `pipe_ops` as `const` (not `pub`). You will need to make `pipe_ops` public in `src/kernel/fs/pipe.zig` (change `const pipe_ops` to `pub const pipe_ops`) AND make `PipeHandle`, `Pipe`, `PipeEnd`, and `PIPE_BUF_SIZE` public so splice.zig can access the internal pipe buffer directly.

Alternatively, if making those public causes import issues (pipe.zig uses `@import("fd")` which is a build.zig module name), add helper functions to pipe.zig:
- `pub fn isPipe(fd: *fd_mod.FileDescriptor) bool` -- returns `fd.ops == &pipe_ops`
- `pub fn getPipeHandle(fd: *fd_mod.FileDescriptor) ?*PipeHandle` -- returns typed cast of private_data if isPipe
- `pub fn getPipeBuffer(handle: *PipeHandle) *Pipe` -- returns the pipe pointer
- `pub fn readFromPipeBuffer(pipe: *Pipe, buf: []u8) usize` -- reads from circular buffer under lock, wakes writers
- `pub fn writeToPipeBuffer(pipe: *Pipe, buf: []const u8) usize` -- writes to circular buffer under lock, wakes readers
- `pub fn peekPipeBuffer(pipe: *Pipe, buf: []u8) usize` -- reads WITHOUT consuming (for tee), under lock

Choose whichever approach produces cleaner code. The helper function approach is safer since it keeps pipe internals encapsulated.

**sys_splice(fd_in: usize, off_in_ptr: usize, fd_out: usize, off_out_ptr: usize, len: usize, flags: usize) SyscallError!usize**
- Linux syscall 275 (x86_64) / 76 (aarch64), already in UAPI
- Exactly one of fd_in or fd_out must be a pipe (EINVAL if neither or both are pipes)
- flags: Accept SPLICE_F_MOVE (1) and SPLICE_F_NONBLOCK (2) -- both can be no-ops since we copy anyway. SPLICE_F_MORE (4) and SPLICE_F_GIFT (8) also accepted as no-ops. Reject unknown flags with EINVAL.
- off_in_ptr/off_out_ptr: If non-zero, pointer to u64 offset for the non-pipe FD. If zero, use FD's current position. It is EINVAL to pass an offset for a pipe end.
- Implementation: Allocate kernel buffer (64KB, like sendfile). In a loop:
  - If fd_in is pipe: read from pipe buffer, write to fd_out at offset
  - If fd_out is pipe: read from fd_in at offset, write to pipe buffer
  - Update offsets, handle partial transfers, stop on EOF or requested length
- IMPORTANT: The pipe end must be the write end when fd_out is a pipe, and the read end when fd_in is a pipe. Check PipeHandle.end field. Return EBADF if wrong end.

**sys_tee(fd_in: usize, fd_out: usize, len: usize, flags: usize) SyscallError!usize**
- Linux syscall 276 (x86_64) / 77 (aarch64)
- Both fd_in and fd_out must be pipe read-end and pipe write-end respectively
- Duplicates data from fd_in's pipe to fd_out's pipe WITHOUT consuming the source
- flags: Same as splice (accept, mostly no-op)
- Implementation: Peek at fd_in pipe buffer (don't advance read_pos), copy peeked data to fd_out pipe buffer
- Return bytes copied (limited by min of available data, available space, len)
- fd_in and fd_out must not refer to the same pipe instance (EINVAL)

**sys_vmsplice(fd: usize, iov_ptr: usize, nr_segs: usize, flags: usize) SyscallError!usize**
- Linux syscall 278 (x86_64) / 75 (aarch64)
- fd must be a pipe (write end)
- Copies user memory described by iovec array into the pipe buffer
- Implementation: Copy iovec array from userspace, validate each base+len, copy each segment into pipe buffer
- For our kernel-copy approach: this is essentially writev into a pipe, but going through the splice path
- Max iovec count: 1024 (like writev)
- Return total bytes written to pipe

**sys_copy_file_range(fd_in: usize, off_in_ptr: usize, fd_out: usize, off_out_ptr: usize, len: usize, flags: usize) SyscallError!usize**
- Linux syscall 326 (x86_64) / 285 (aarch64)
- Both fd_in and fd_out must be regular files (not pipes, not sockets)
- flags must be 0 (Linux currently defines no flags, return EINVAL if non-zero)
- off_in_ptr/off_out_ptr: pointers to u64 offsets. If non-zero, use and update that offset. If zero, use and update the FD's current position.
- Both FDs must support read/seek (in) and write/seek (out)
- Implementation: Very similar to sendfile -- allocate 64KB kernel buffer, loop read/write
- fd_in and fd_out may refer to the same file (Linux allows this for in-file copies)
- Return total bytes copied

**Userspace wrappers in src/user/lib/syscall/io.zig:**

Add after the existing `renameat2` function:

```zig
// Zero-copy I/O constants
pub const SPLICE_F_MOVE: u32 = 1;
pub const SPLICE_F_NONBLOCK: u32 = 2;
pub const SPLICE_F_MORE: u32 = 4;
pub const SPLICE_F_GIFT: u32 = 8;

/// Move data between a file descriptor and a pipe (kernel-side copy)
pub fn splice(fd_in: i32, off_in: ?*u64, fd_out: i32, off_out: ?*u64, len: usize, flags: u32) SyscallError!size_t {
    const ret = primitive.syscall6(
        syscalls.SYS_SPLICE,
        @bitCast(@as(isize, fd_in)),
        if (off_in) |p| @intFromPtr(p) else 0,
        @bitCast(@as(isize, fd_out)),
        if (off_out) |p| @intFromPtr(p) else 0,
        len,
        @as(usize, flags),
    );
    if (primitive.isError(ret)) return primitive.errorFromReturn(ret);
    return ret;
}

/// Duplicate pipe data without consuming the source
pub fn tee(fd_in: i32, fd_out: i32, len: usize, flags: u32) SyscallError!size_t {
    const ret = primitive.syscall4(
        syscalls.SYS_TEE,
        @bitCast(@as(isize, fd_in)),
        @bitCast(@as(isize, fd_out)),
        len,
        @as(usize, flags),
    );
    if (primitive.isError(ret)) return primitive.errorFromReturn(ret);
    return ret;
}

/// Splice user pages into a pipe
pub fn vmsplice(fd: i32, iov: []const Iovec, flags: u32) SyscallError!size_t {
    const ret = primitive.syscall4(
        syscalls.SYS_VMSPLICE,
        @bitCast(@as(isize, fd)),
        @intFromPtr(iov.ptr),
        iov.len,
        @as(usize, flags),
    );
    if (primitive.isError(ret)) return primitive.errorFromReturn(ret);
    return ret;
}

/// Copy data between two files within the kernel
pub fn copy_file_range(fd_in: i32, off_in: ?*u64, fd_out: i32, off_out: ?*u64, len: usize, flags: u32) SyscallError!size_t {
    const ret = primitive.syscall6(
        syscalls.SYS_COPY_FILE_RANGE,
        @bitCast(@as(isize, fd_in)),
        if (off_in) |p| @intFromPtr(p) else 0,
        @bitCast(@as(isize, fd_out)),
        if (off_out) |p| @intFromPtr(p) else 0,
        len,
        @as(usize, flags),
    );
    if (primitive.isError(ret)) return primitive.errorFromReturn(ret);
    return ret;
}
```
