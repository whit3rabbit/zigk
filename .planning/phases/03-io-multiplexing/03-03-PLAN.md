```zig
pub fn sys_pselect6(nfds: usize, readfds: usize, writefds: usize, exceptfds: usize, timeout_ptr: usize, sigmask_ptr: usize) SyscallError!usize {
```

Implementation:
1. If sigmask_ptr != 0:
   - Read the pselect6 sigmask argument. On Linux, the 6th arg is a pointer to `struct { sigset_t *ss; size_t ss_len; }`, NOT the sigset directly. Read this struct from userspace.
   - Validate ss_len == 8 (size of u64 sigset)
   - Read the signal mask from ss pointer
   - Save current thread's sigmask, apply new mask: `thread.sigmask = new_mask`
   - Use defer to restore: `defer thread.sigmask = old_mask`

2. Convert timespec to timeval for select-compatible timeout:
   - If timeout_ptr != 0: read Timespec from userspace, convert to microseconds, create a local timeval-style timeout
   - If timeout_ptr == 0: pass 0 as timeout (infinite wait)

3. Delegate to the existing sys_select logic. To avoid code duplication, extract the core select loop into a helper function `selectInternal(nfds, readfds, writefds, exceptfds, timeout_us: ?u64) SyscallError!usize` and have both sys_select and sys_pselect6 call it.

   The selectInternal function takes the parsed timeout in microseconds (null = infinite) and does the existing select poll loop.

   Refactor sys_select to:
   - Parse timeval timeout as before
   - Call selectInternal

**Step 3: Upgrade sys_ppoll to actually poll file descriptors.**

Replace the stub body of sys_ppoll (around line 251) with real fd polling:

1. Keep existing argument validation (sigmask size, timeout parsing)
2. If sigmask_ptr != 0: apply signal mask temporarily (same pattern as pselect6)
3. Copy pollfd array from userspace to kernel buffer (same as sys_poll in net/poll.zig)
4. Poll loop:
   - For each pollfd with fd >= 0:
     - Look up fd in FdTable
     - If not found: set revents = POLLNVAL
     - If found and ops.poll exists: call it, convert u32 result to i16 revents
     - If found but no poll: assume ready based on isReadable/isWritable
   - If any revents != 0 or timeout == 0: break
   - Else: yield and retry, checking timeout
5. Copy results back to userspace
6. Return ready count

This is essentially the same logic as the upgraded sys_poll but with timespec timeout and signal mask.

In `src/kernel/sys/syscall/net/poll.zig`, the current sys_poll only checks sockets (via getSocketContext/checkPollEvents). Upgrade it to use FileOps.poll for ALL fd types:

In the poll check loop (around line 134), replace the socket-only logic:

Current pattern (per fd):
```
if fd <= 2: hardcoded stdout check
else: getSocketContext -> checkPollEvents
```

New pattern (per fd):
```
if pfd.fd < 0: continue (skip negative fds)
Look up fd in FdTable: base.getGlobalFdTable().get(fd_u32)
if not found: revents = POLLNVAL
if found and fd.ops.poll exists:
    poll_result = fd.ops.poll(fd, @as(u32, @intCast(@as(u16, @bitCast(pfd.events)))))
    revents = @truncate(poll_result)  // u32 -> i16 for PollFd.revents
if found but no poll:
    assume ready based on isReadable/isWritable (POLLIN if readable, POLLOUT if writable)
Set pfd.revents = revents
if revents != 0: ready_count += 1
```

Do the same for the re-check loop after blocking (around line 230).

Remove the getSocketContext dependency for the poll check. The socket_file_ops parameter to sys_poll may still be needed for backward compat, but the poll dispatch should go through FileOps.poll universally. Socket FDs now have poll methods wired (unixSocketPoll/unixSocketFullPoll already exist), and TCP sockets can be handled via their FileOps if present, or fall back to the old checkPollEvents path for fds without FileOps.poll.

**Step 2: Add userspace wrappers in io.zig.**

```zig
// =============================================================================
// I/O Multiplexing - epoll
// =============================================================================

pub fn epoll_create1(flags: u32) SyscallError!i32 {
    const ret = primitive.syscall1(syscalls.SYS_EPOLL_CREATE1, @as(usize, flags));
    if (primitive.isError(ret)) return primitive.errorFromReturn(ret);
    return @intCast(ret);
}

pub fn epoll_ctl(epfd: i32, op: u32, fd: i32, event: ?*EpollEvent) SyscallError!usize {
    const ev_ptr: usize = if (event) |e| @intFromPtr(e) else 0;
    const ret = primitive.syscall4(
        syscalls.SYS_EPOLL_CTL,
        @bitCast(@as(isize, epfd)),
        @as(usize, op),
        @bitCast(@as(isize, fd)),
        ev_ptr,
    );
    if (primitive.isError(ret)) return primitive.errorFromReturn(ret);
    return ret;
}

pub fn epoll_wait(epfd: i32, events: [*]EpollEvent, maxevents: u32, timeout: i32) SyscallError!usize {
    const ret = primitive.syscall4(
        syscalls.SYS_EPOLL_WAIT,
        @bitCast(@as(isize, epfd)),
        @intFromPtr(events),
        @as(usize, maxevents),
        @bitCast(@as(isize, timeout)),
    );
    if (primitive.isError(ret)) return primitive.errorFromReturn(ret);
    return ret;
}

pub const EpollEvent = uapi.epoll.EpollEvent;
pub const EPOLL_CTL_ADD = uapi.epoll.EPOLL_CTL_ADD;
pub const EPOLL_CTL_DEL = uapi.epoll.EPOLL_CTL_DEL;
pub const EPOLL_CTL_MOD = uapi.epoll.EPOLL_CTL_MOD;
pub const EPOLLIN = uapi.epoll.EPOLLIN;
pub const EPOLLOUT = uapi.epoll.EPOLLOUT;
pub const EPOLLERR = uapi.epoll.EPOLLERR;
pub const EPOLLHUP = uapi.epoll.EPOLLHUP;
pub const EPOLLET = uapi.epoll.EPOLLET;
pub const EPOLLONESHOT = uapi.epoll.EPOLLONESHOT;

// =============================================================================
// I/O Multiplexing - select/pselect6
// =============================================================================

pub fn select(nfds: i32, readfds: ?*[128]u8, writefds: ?*[128]u8, exceptfds: ?*[128]u8, timeout: ?*extern struct { tv_sec: i64, tv_usec: i64 }) SyscallError!usize {
    const ret = primitive.syscall5(
        syscalls.SYS_SELECT,
        @bitCast(@as(isize, nfds)),
        if (readfds) |p| @intFromPtr(p) else 0,
        if (writefds) |p| @intFromPtr(p) else 0,
        if (exceptfds) |p| @intFromPtr(p) else 0,
        if (timeout) |p| @intFromPtr(p) else 0,
    );
    if (primitive.isError(ret)) return primitive.errorFromReturn(ret);
    return ret;
}
```
