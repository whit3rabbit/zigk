**Tests to implement:**

1. **testEpollCreateAndClose** - Create epoll instance with epoll_create1(0), verify returns valid fd (>= 0), close it.

2. **testEpollCtlAddAndWait** - Create pipe, create epoll, add pipe read end with EPOLLIN, write data to pipe write end, call epoll_wait with timeout=0. Verify returns 1 event with EPOLLIN set.

3. **testEpollWaitNoEvents** - Create epoll (empty, no fds added), call epoll_wait with timeout=0. Verify returns 0 (no events).

4. **testEpollPipeHup** - Create pipe, add read end to epoll with EPOLLIN, close write end, call epoll_wait. Verify returns EPOLLHUP (and possibly EPOLLIN).

5. **testEpollRegularFileAlwaysReady** - Open a regular file (e.g., /etc/hostname from initrd), add to epoll with EPOLLIN|EPOLLOUT, epoll_wait with timeout=0. Verify returns with both EPOLLIN and EPOLLOUT.

6. **testSelectPipeReadable** - Create pipe, write data, use select to check read fd. Verify readfds bit is set on return.

7. **testSelectPipeWritable** - Create pipe (empty), use select to check write fd for writability. Verify writefds bit is set.

8. **testSelectTimeout** - Call select with no ready fds and timeout=0. Verify returns 0 immediately.

9. **testPollPipeEvents** - Create pipe, write data, call poll on read end with POLLIN. Verify revents has POLLIN set.

10. **testPollPipeHup** - Create pipe, close write end, call poll on read end. Verify POLLHUP is returned.

**Implementation notes:**
- For pipe tests: use `io.pipe()` or the equivalent userspace syscall wrapper to create pipes
- For select: construct fd_set bitmasks manually (128-byte arrays, set bit fd%8 in byte fd/8)
- For epoll: use the new epoll_create1/epoll_ctl/epoll_wait wrappers from io.zig
- For poll: use the existing poll wrapper from io.zig
- Close all fds after each test to prevent fd leaks across tests
- Use timeout=0 for all blocking calls to prevent test hangs
- Each test should be self-contained (create resources, test, cleanup)

**fd_set helper pattern for select tests:**
```zig
fn fdSet(set: *[128]u8, fd: i32) void {
    const ufd: u32 = @intCast(fd);
    set[ufd / 8] |= @as(u8, 1) << @truncate(ufd % 8);
}
fn fdIsSet(set: *const [128]u8, fd: i32) bool {
    const ufd: u32 = @intCast(fd);
    return (set[ufd / 8] & (@as(u8, 1) << @truncate(ufd % 8))) != 0;
}
```
