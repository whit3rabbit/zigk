**Part B: Add invalidateCacheEntry to inode.zig**

Add a public helper to `src/fs/ext2/inode.zig` that alloc.zig can call after zeroInode to prevent stale cache entries:

```zig
/// Invalidate any inode cache entry matching `inum`.
///
/// Called after zeroInode to prevent stale reads from the LRU cache.
/// Scan is O(INODE_CACHE_SIZE) = O(16) -- negligible.
/// The inode will be re-read from disk on the next getCachedInode call.
pub fn invalidateCacheEntry(fs: *Ext2Fs, inum: u32) void {
    for (&fs.inode_cache) |*entry| {
        if (entry.inum == inum) {
            entry.inum = 0; // Mark slot as empty
            return; // Only one entry per inum in the cache
        }
    }
}
```

**Function 1: allocBlock (ALLOC-01)**

Implement `pub fn allocBlock(fs: *Ext2Fs) AllocError!u32` following 49-RESEARCH.md Pattern 1 exactly:

Phase 1 (under alloc_lock):
- Walk groups from group 0 to group_count-1 looking for one with `bg_free_blocks_count > 0`
- For each candidate group:
  - Heap-allocate a bitmap buffer (fs.block_size bytes) via `heap.allocator().alloc(u8, fs.block_size)`
  - `@memset(bitmap_buf, 0)` for DMA hygiene
  - Read the bitmap block from `gd.bg_block_bitmap` via `fs.dev.readSectors`
    - SAFE: readSectors is synchronous in current VirtIO-SCSI/AHCI drivers. alloc_lock IS held during this read to prevent TOCTOU with concurrent allocators. Do not hold alloc_lock here if async I/O is added.
  - Scan for first zero bit: outer loop by byte (skip 0xFF bytes), inner loop by bit
  - On finding a free bit:
    - Set the bit in the buffer
    - Compute absolute block number: `group_idx * s_blocks_per_group + local_block + s_first_data_block`
    - Use `std.math.mul(u32, ...)` and `std.math.add(u32, ...)` for all arithmetic
    - Validate `result_block < s_blocks_count`, undo bit if not
    - Decrement `gd.bg_free_blocks_count` and `fs.superblock.s_free_blocks_count` (saturating sub for safety)
    - Record result_block, result_group, mark found=true, break
  - If not found in this group, free bitmap_buf and continue to next group
- If no group has free blocks, return error.ENOSPC

Phase 2 (outside alloc_lock):
- `defer heap.allocator().free(bitmap_buf)` for the final bitmap buffer
- Write bitmap block to disk via `fs.dev.writeSectors`
- On write failure: rollback in-memory counters under lock, return error
- Call `writeGroupDescriptor(fs, result_group)`
- Call `writeSuperblock(fs)`
- Return result_block

IMPORTANT NOTES:
- Bitmap buffer MUST be heap-allocated (4KB on kernel stack overflows aarch64 per MEMORY.md)
- Free bitmap_buf for failed groups INSIDE the group scan loop (Pitfall 4 from research)
- The `bg_free_blocks_count` field on GroupDescriptor is u16, so use saturating arithmetic: `-|= 1` for decrement, `+|= 1` for increment

**Function 2: freeBlock (ALLOC-02)**

Implement `pub fn freeBlock(fs: *Ext2Fs, block_num: u32) AllocError!void` following 49-RESEARCH.md Pattern 2:

- Validate: `block_num >= s_first_data_block` and `block_num < s_blocks_count`
- Compute group_idx, local_idx, byte_idx, bit_idx from block_num
- Read bitmap outside lock (heap-allocate buffer, zero, readSectors)
- Warn (do not panic) if bit is already zero (double-free)
- Clear the bit in the buffer
- Write bitmap outside lock
- Update in-memory counters under alloc_lock: increment `gd.bg_free_blocks_count` and `fs.superblock.s_free_blocks_count`
- Flush group descriptor and superblock outside lock

**Function 3: allocInode (ALLOC-03)**

Implement `pub fn allocInode(fs: *Ext2Fs) AllocError!u32` with the same two-phase structure as allocBlock, with these differences:
- Read `gd.bg_inode_bitmap` instead of `gd.bg_block_bitmap`
- Result inode number = `group_idx * s_inodes_per_group + local_idx + 1` (1-based)
- Validate: `result_inum > 0` and `result_inum <= s_inodes_count`
- Decrement `gd.bg_free_inodes_count` and `fs.superblock.s_free_inodes_count`
- After flushing bitmap + group desc + superblock, call `zeroInode(fs, result_inum)` to clear stale data
- After zeroInode, call `inode_mod.invalidateCacheEntry(fs, result_inum)` to prevent stale cache reads

**Function 4: freeInode (ALLOC-04)**

Implement `pub fn freeInode(fs: *Ext2Fs, inum: u32) AllocError!void` with the same structure as freeBlock, with these differences:
- Convert inum to group: `group_idx = (inum - 1) / s_inodes_per_group`, `local_idx = (inum - 1) % s_inodes_per_group`
- Read/clear `gd.bg_inode_bitmap`
- Increment `gd.bg_free_inodes_count` and `fs.superblock.s_free_inodes_count`
- After flushing, call `zeroInode(fs, inum)` to prevent data leaks from the freed inode
- After zeroInode, call `inode_mod.invalidateCacheEntry(fs, inum)` to prevent stale cache reads

**Helper: writeGroupDescriptor**

Implement `pub fn writeGroupDescriptor(fs: *Ext2Fs, group_idx: u32) AllocError!void` following 49-RESEARCH.md Pattern 3:
- BGDT block = `s_first_data_block + 1`
- Compute byte offset of the group descriptor within the BGDT
- Read the containing sector, overlay the updated GroupDescriptor bytes, write back
- Use `std.mem.asBytes` to serialize the in-memory GroupDescriptor

**Helper: writeSuperblock**

Implement `pub fn writeSuperblock(fs: *Ext2Fs) AllocError!void` following 49-RESEARCH.md Pattern 3:
- Superblock is always at SUPERBLOCK_LBA (2), occupies 2 sectors (1024 bytes)
- Copy in-memory superblock to a 1024-byte buffer, write via writeSectors

**Helper: zeroInode**

Implement `fn zeroInode(fs: *Ext2Fs, inum: u32) AllocError!void` following 49-RESEARCH.md Pattern 4:
- Compute the inode table LBA using the same formula as `readInode` in inode.zig
- Read 2 sectors (to handle inode spanning a sector boundary)
- Zero the `@sizeOf(types.Inode)` bytes at the correct offset within the buffer
- Write 2 sectors back
- Zero-init the sector buffer before read (DMA hygiene)

**Function 8: selfTest (end-to-end kernel-side verification)**

Implement `pub fn selfTest(fs: *Ext2Fs) void` that exercises the full alloc/free round-trip:

```zig
/// Kernel-side self-test for allocation primitives.
///
/// Runs allocBlock+freeBlock and allocInode+freeInode, verifying that
/// superblock free counters return to their baseline values after each
/// round-trip. Called from initExt2Fs after mounting.
///
/// This function logs PASSED or FAILED to the console. It does NOT
/// panic on failure -- a failed self-test is an error, not a crash.
pub fn selfTest(fs: *Ext2Fs) void {
    const blocks_before = fs.superblock.s_free_blocks_count;
    const inodes_before = fs.superblock.s_free_inodes_count;

    // --- Block alloc/free round-trip ---
    const block_num = allocBlock(fs) catch |err| {
        console.err("ext2 selfTest: allocBlock failed: {}", .{err});
        return;
    };
    // Verify counter decremented.
    if (fs.superblock.s_free_blocks_count != blocks_before - 1) {
        console.err("ext2 selfTest: FAILED -- allocBlock did not decrement s_free_blocks_count", .{});
        return;
    }
    freeBlock(fs, block_num) catch |err| {
        console.err("ext2 selfTest: freeBlock failed: {}", .{err});
        return;
    };
    if (fs.superblock.s_free_blocks_count != blocks_before) {
        console.err("ext2 selfTest: FAILED -- freeBlock did not restore s_free_blocks_count", .{});
        return;
    }

    // --- Inode alloc/free round-trip ---
    const inum = allocInode(fs) catch |err| {
        console.err("ext2 selfTest: allocInode failed: {}", .{err});
        return;
    };
    if (fs.superblock.s_free_inodes_count != inodes_before - 1) {
        console.err("ext2 selfTest: FAILED -- allocInode did not decrement s_free_inodes_count", .{});
        return;
    }
    freeInode(fs, inum) catch |err| {
        console.err("ext2 selfTest: freeInode failed: {}", .{err});
        return;
    };
    if (fs.superblock.s_free_inodes_count != inodes_before) {
        console.err("ext2 selfTest: FAILED -- freeInode did not restore s_free_inodes_count", .{});
        return;
    }

    console.info("ext2 selfTest: PASSED (block={d}, inode={d}, counts restored)", .{ block_num, inum });
}
```
