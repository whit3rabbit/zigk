<objective>
Implement sys_readv, sys_pwrite64, sys_preadv, and sys_pwritev -- the core vectored and positional I/O syscalls.

Purpose: These syscalls enable scatter-gather I/O patterns used by databases (SQLite, Postgres) and file servers. readv mirrors the existing writev. pwrite64 mirrors pread64. preadv/pwritev combine vectored I/O with positional access.

Output: Four new syscall implementations in read_write.zig, exported via root.zig for dispatch table auto-registration.
</objective>

  <files>src/kernel/sys/syscall/io/read_write.zig</files>
  <action>
Add two new syscall functions to read_write.zig after the existing sys_writev:

**sys_readv(fd: usize, bvec_ptr: usize, count: usize) SyscallError!usize:**
- Mirror the existing sys_writev pattern exactly but for reads
- Validate count == 0 returns 0, count > 1024 returns EINVAL
- Allocate kvecs array from heap, copy iovecs from user via UserPtr.copyToKernel
- Validate total iov_len sum with @addWithOverflow, cap at MAX_READV_BYTES = 16 * 1024 * 1024
- Get fd from table, check isReadable(), check ops.read != null (return EBADF if not readable)
- Acquire fd.lock for the entire operation (atomicity -- no interleaving)
- Loop over kvecs: for each vec with len > 0, chunk into 64KB pieces
- Use utils.perform_read_locked(fd_obj, current_base, chunk_len) for each chunk
- Accumulate total_read with @addWithOverflow
- On short read (res < chunk_len), return total_read immediately
- On error, if total_read > 0 return total_read, else propagate error

**sys_pwrite64(fd_num: usize, buf_ptr: usize, count: usize, offset: usize) SyscallError!usize:**
- Mirror the existing sys_pread64 pattern exactly but for writes
- Validate count == 0, get fd, check isWritable(), check ops.write and ops.seek not null (ESPIPE)
- Acquire fd.lock
- Save old_pos = fd.position
- Seek to offset via seek_fn(fd, @intCast(offset), 0) -- SEEK_SET
- Use utils.perform_write_locked(fd, buf_ptr, count) for the write
- Restore position via seek_fn(fd, @intCast(old_pos), 0) on both success and error (use pattern from pread64)
- Return bytes_written

Use the same import aliases already at the top of the file (base, utils, heap, etc.). Define the Iovec struct once at module level rather than duplicating inside each function -- extract the existing Iovec from sys_writev to module scope and reuse it in sys_readv and the preadv/pwritev functions.

Add two new syscall functions to read_write.zig after sys_pwrite64:

**sys_preadv(fd_num: usize, bvec_ptr: usize, count: usize, offset: usize) SyscallError!usize:**
- Combines sys_readv (vectored) with sys_pread64 (positional)
- Validate count (0 returns 0, >1024 returns EINVAL)
- Copy iovecs from user, validate total length with overflow check
- Get fd, check isReadable(), check ops.read and ops.seek not null (ESPIPE if no seek)
- Acquire fd.lock
- Save old_pos, seek to offset
- Loop over kvecs performing reads via perform_read_locked (same chunking as readv)
- On completion or short read, restore position via seek_fn before returning
- On error mid-read, restore position before returning (use errdefer-like pattern from pread64)
- Return total bytes read

**sys_pwritev(fd_num: usize, bvec_ptr: usize, count: usize, offset: usize) SyscallError!usize:**
- Combines sys_writev (vectored) with sys_pwrite64 (positional)
- Same validation as preadv but check isWritable() and ops.write not null
- Save position, seek to offset, loop over kvecs writing via perform_write_locked
- Restore position on success and error
- Return total bytes written

Both functions should share the iovec copying and validation pattern from readv/writev. The position save/seek/restore pattern should match pread64 exactly.
