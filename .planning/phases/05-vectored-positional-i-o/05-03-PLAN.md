Add the following wrappers to io.zig in the "Advanced I/O Syscalls" section or a new "Vectored & Positional I/O" section. The existing Iovec type and writev wrapper already exist. Follow the exact pattern used by the existing writev, pread64, and other wrappers.

**readv(fd: i32, iov: []const Iovec) SyscallError!size_t:**
```zig
pub fn readv(fd: i32, iov: []const Iovec) SyscallError!size_t {
    const ret = primitive.syscall3(
        syscalls.SYS_READV,
        @bitCast(@as(isize, fd)),
        @intFromPtr(iov.ptr),
        iov.len,
    );
    if (primitive.isError(ret)) return primitive.errorFromReturn(ret);
    return ret;
}
```

Note: readv uses Iovec with `base: usize` (pointer as integer). For mutable read buffers, user creates Iovec with @intFromPtr of mutable buffer. This matches the existing writev pattern.

**pwrite64(fd: i32, buf: [*]const u8, count: usize, offset: u64) SyscallError!size_t:**
- syscall4 with SYS_PWRITE64, fd, buf ptr, count, offset

**preadv(fd: i32, iov: []const Iovec, offset: u64) SyscallError!size_t:**
- syscall4 with SYS_PREADV, fd, iov ptr, iov.len, offset

**pwritev(fd: i32, iov: []const Iovec, offset: u64) SyscallError!size_t:**
- syscall4 with SYS_PWRITEV, fd, iov ptr, iov.len, offset

**preadv2(fd: i32, iov: []const Iovec, offset: i64, flags: u32) SyscallError!size_t:**
- syscall6 with SYS_PREADV2, fd, iov ptr, iov.len, offset low 32 bits, offset high 32 bits, flags
- Split offset: `const off_u: u64 = @bitCast(offset);` then `off_u & 0xFFFFFFFF` and `off_u >> 32`

**pwritev2(fd: i32, iov: []const Iovec, offset: i64, flags: u32) SyscallError!size_t:**
- Same pattern as preadv2 but with SYS_PWRITEV2

**sendfile(out_fd: i32, in_fd: i32, offset: ?*u64, count: usize) SyscallError!size_t:**
- syscall4 with SYS_SENDFILE, out_fd, in_fd, offset ptr (or 0 if null), count

Also add RWF_* flag constants for userspace:
```zig
pub const RWF_HIPRI: u32 = 0x00000001;
pub const RWF_DSYNC: u32 = 0x00000002;
pub const RWF_SYNC: u32 = 0x00000004;
pub const RWF_NOWAIT: u32 = 0x00000008;
pub const RWF_APPEND: u32 = 0x00000010;
```

**Test 1: testReadvBasic** (VIO-01)
- Open /shell.elf O_RDONLY
- Create 2 Iovec buffers: buf1[4] for ELF magic, buf2[12] for rest of header
- Call readv with both iovecs
- Verify total bytes >= 16
- Verify buf1 contains ELF magic (0x7F, 'E', 'L', 'F')

**Test 2: testReadvEmptyVec** (VIO-01 edge case)
- Open /shell.elf O_RDONLY
- Call readv with count=0 iovecs
- Should return 0 (not error)

**Test 3: testWritevReadv** (VIO-01 + VIO-02 roundtrip)
- Create /mnt/test_vectored.txt O_RDWR|O_CREAT|O_TRUNC (skip if SFS unavailable)
- writev two buffers: "Hello" and " World"
- lseek to 0
- readv into two buffers: buf1[5] and buf2[6]
- Verify buf1 == "Hello" and buf2 == " World"

**Test 4: testPreadvBasic** (VIO-03)
- Open /shell.elf O_RDONLY
- Read first 4 bytes normally to advance position
- Call preadv with offset=0, one iovec for 4 bytes
- Verify preadv returns ELF magic (offset 0, not current position)
- Verify file position unchanged (read 1 more byte, should be 5th byte not 1st)

**Test 5: testPwritevBasic** (VIO-04)
- Create /mnt/test_pwritev.txt O_RDWR|O_CREAT|O_TRUNC (skip if SFS unavailable)
- Write "AAAA" at position 0
- pwritev "BB" at offset 1 (overwrite middle)
- Verify position unchanged (should still be 4)
- lseek to 0, read all: should be "ABBA" (A + BB + A)

- pwritev at offset 0: write "Hello" + ", " (7 bytes total)
- lseek to 0, read back, verify "Hello, A" -- no, pwritev writes 7 bytes so "Hello, " replaces all. Verify position was not changed by pwritev.
  Simpler test: write "1234567890", pwritev "XX" at offset 3, read back "123XX67890", verify file position was unchanged.

**Test 6: testPreadv2FlagsZero** (VIO-05)
- Same as preadv test but using preadv2 with flags=0 and explicit offset
- Should behave identically to preadv

**Test 7: testPwritev2FlagsZero** (VIO-06)
- Same as pwritev test but using pwritev2 with flags=0
- Should behave identically to pwritev

**Test 8: testPreadv2OffsetNeg1** (VIO-05 offset=-1 mode)
- Open /shell.elf O_RDONLY
- lseek to 0
- Call preadv2 with offset=-1 (use current position), flags=0
- Should read from current position (offset 0), returning ELF magic

**Test 9: testPreadv2HipriFlag** (VIO-05 RWF_HIPRI)
- Open /shell.elf O_RDONLY
- Call preadv2 with RWF_HIPRI flag
- Should return EOPNOTSUPP (OperationNotSupported error)

**Test 10: testSendfileBasic** (VIO-07)

- Call sendfile(pipe_write_fd, in_fd, null, 64)
- Read from pipe_read_fd, verify first 4 bytes are ELF magic
- Verify sendfile returned bytes transferred (should be 64)

**Test 11: testSendfileWithOffset** (VIO-07 with offset)
- Open /shell.elf O_RDONLY
- Create pipe
- var offset: u64 = 0
- Call sendfile(pipe_write, in_fd, &offset, 16)
- Verify offset was updated to 16
- Verify in_fd position unchanged (lseek SEEK_CUR should return 0)

**Test 12: testSendfileInvalidFd** (VIO-07 error)
