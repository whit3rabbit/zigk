  ```zig
  /// Set time of day (requires root)
  pub fn settimeofday(tv: *const Timeval) SyscallError!void {
      const ret = primitive.syscall2(syscalls.SYS_SETTIMEOFDAY, @intFromPtr(tv), 0);
      if (primitive.isError(ret)) return primitive.errorFromReturn(ret);
  }
  ```

  - `testRtSigsuspendBasic`: Block SIGUSR1 via sigprocmask. Send SIGUSR1 to self (pending). Call rt_sigsuspend with a mask that does NOT include SIGUSR1 (unblocking it). This should deliver the signal and return EINTR. Verify the handler ran. Note: rt_sigsuspend always returns EINTR per POSIX. Use the existing sigusr1_called flag pattern. The test must: (1) install handler, (2) block SIGUSR1, (3) send SIGUSR1 to self (queued), (4) call rt_sigsuspend with empty mask (allowing SIGUSR1), (5) verify handler ran and call returned error.Interrupted.

  Careful: The userspace wrapper `rt_sigsuspend` maps EINTR to `error.Interrupted` (check primitive.zig error mapping). If it maps to a different name, use that.
