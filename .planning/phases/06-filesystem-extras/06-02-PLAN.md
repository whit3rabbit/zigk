<objective>
Implement sys_utimensat and sys_futimesat syscalls with full POSIX semantics, plus userspace wrappers.

Purpose: utimensat enables nanosecond-precision timestamp control used by tar, rsync, cp -p, and make. futimesat is the legacy microsecond wrapper required for backward compatibility. These are the last two syscalls needed for Phase 6 filesystem completeness.
Output: Working utimensat/futimesat dispatched via the auto-discovery table, with userspace wrappers callable from test harness and shell.
</objective>

**Function signature:**
```zig
pub fn sys_utimensat(dirfd: usize, path_ptr: usize, times_ptr: usize, flags: usize) base.SyscallError!usize
```

**Implementation details:**

1. **Flag validation:**
   - Define `const AT_SYMLINK_NOFOLLOW: usize = 0x100;`
   - If `flags & AT_SYMLINK_NOFOLLOW != 0`, return `error.ENOSYS` (MVP: symlink timestamp modification not supported)
   - If any other flags set (`flags & ~AT_SYMLINK_NOFOLLOW != 0`), return `error.EINVAL`

2. **Define special timespec values:**
   - `const UTIME_NOW: i64 = (1 << 30) - 1;` (0x3fffffff)
   - `const UTIME_OMIT: i64 = (1 << 30) - 2;` (0x3ffffffe)

3. **Read timespec array from userspace:**
   - If `times_ptr == 0` (NULL): set both to UTIME_NOW semantics
   - Otherwise: validate access with `isValidUserAccess(times_ptr, @sizeOf([2]uapi.Timespec), .Read)`, read via `UserPtr.from(times_ptr).readValue([2]uapi.Timespec)`
   - `times[0]` = atime, `times[1]` = mtime (POSIX convention)

4. **Validate nsec values:**
   - For each timespec: if `tv_nsec != UTIME_NOW and tv_nsec != UTIME_OMIT`, check `tv_nsec >= 0 and tv_nsec <= 999_999_999`, else return `error.EINVAL`

5. **Resolve atime and mtime values:**
   - Use `hal.timing` for current time when UTIME_NOW is specified. Import `hal` at the top of the file. Use pattern from timerfd.zig: `const now_ns = hal.timing.getClockNanoseconds(.realtime);` then `const now_sec = @as(i64, @intCast(now_ns / 1_000_000_000));` and `const now_nsec = @as(i64, @intCast(now_ns % 1_000_000_000));`
   - For UTIME_OMIT: pass -1 as the sec value to Vfs.setTimestamps (convention from Plan 01)
   - For UTIME_NOW: compute current time, use those values
   - For normal values: use tv_sec and tv_nsec directly

6. **Path resolution:**
   - Copy path from userspace via `copyStringFromUser`
   - If absolute path (`path[0] == '/'`): use directly
   - If relative: resolve via `fd_syscall.resolvePathAt(dirfd, path, resolved_buf)`
   - Canonicalize the final path

7. **Call Vfs.setTimestamps:**
   - Map VFS errors: `NotSupported -> EROFS`, `NotFound -> ENOENT`, `IOError -> EIO`
   - Return 0 on success

**Also add `sys_futimesat`:**

- If `times_ptr == 0`: delegate to `sys_utimensat(dirfd, path_ptr, 0, 0)` (NULL times = set both to now)

- Validate: `tv_usec >= 0 and tv_usec < 1_000_000`, else return `error.EINVAL`
- Write the converted `[2]Timespec` to a stack variable, then call the internal utimensat logic (extract an internal helper if needed, or pass the resolved timespec directly to Vfs.setTimestamps using the same path resolution logic)

**1. In `src/user/lib/syscall/io.zig`:**

```zig
/// Timespec for utimensat (matches kernel uapi.abi.Timespec)
pub const Timespec = extern struct {
    tv_sec: i64,
    tv_nsec: i64,
};

/// Timeval for futimesat (microsecond precision)
pub const Timeval = extern struct {
    tv_sec: i64,
    tv_usec: i64,
};

/// Special tv_nsec values for utimensat
pub const UTIME_NOW: i64 = (1 << 30) - 1;
pub const UTIME_OMIT: i64 = (1 << 30) - 2;

/// Set file timestamps with nanosecond precision relative to directory FD
/// times: pointer to [2]Timespec (atime, mtime), or null to set both to current time
/// flags: 0 or AT_SYMLINK_NOFOLLOW (0x100)
pub fn utimensat(dirfd: i32, path: [*:0]const u8, times: ?*const [2]Timespec, flags: i32) SyscallError!void {
    const times_ptr: usize = if (times) |t| @intFromPtr(t) else 0;
    const ret = primitive.syscall4(syscalls.SYS_UTIMENSAT, @bitCast(@as(isize, dirfd)), @intFromPtr(path), times_ptr, @bitCast(@as(isize, flags)));
    if (primitive.isError(ret)) return primitive.errorFromReturn(ret);
}

/// Set file timestamps with microsecond precision relative to directory FD (legacy)
/// times: pointer to [2]Timeval (atime, mtime), or null to set both to current time
pub fn futimesat(dirfd: i32, path: [*:0]const u8, times: ?*const [2]Timeval) SyscallError!void {
    const times_ptr: usize = if (times) |t| @intFromPtr(t) else 0;
    const ret = primitive.syscall3(syscalls.SYS_FUTIMESAT, @bitCast(@as(isize, dirfd)), @intFromPtr(path), times_ptr);
    if (primitive.isError(ret)) return primitive.errorFromReturn(ret);
}
```
