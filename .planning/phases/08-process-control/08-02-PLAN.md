1. Add to `src/user/lib/syscall/process.zig` at the end (before any closing brace), in a new section:

   ```
   // =============================================================================
   // Process Control (prctl, CPU affinity)
   // =============================================================================
   ```

   **prctl(option, arg2, arg3, arg4, arg5) -> SyscallError!usize:**
   - `const ret = primitive.syscall5(syscalls.SYS_PRCTL, option, arg2, arg3, arg4, arg5);`
   - Check error, return value on success.

   **PR_SET_NAME and PR_GET_NAME constants:**
   - `pub const PR_SET_NAME: usize = 15;`
   - `pub const PR_GET_NAME: usize = 16;`

   **sched_setaffinity(pid, cpusetsize, mask_ptr) -> SyscallError!void:**
   - `const ret = primitive.syscall3(syscalls.SYS_SCHED_SETAFFINITY, @as(usize, @as(u32, @bitCast(pid))), cpusetsize, @intFromPtr(mask_ptr));`
   - pid is i32, cast to usize via u32 bitcast (standard pattern for pid args, see setpgid).

   **sched_getaffinity(pid, cpusetsize, mask_ptr) -> SyscallError!usize:**
   - `const ret = primitive.syscall3(syscalls.SYS_SCHED_GETAFFINITY, @as(usize, @as(u32, @bitCast(pid))), cpusetsize, @intFromPtr(mask_ptr));`
   - Returns the number of bytes written to mask (kernel's cpuset size).

   Note: SYS_PRCTL already exists in syscalls root. SYS_SCHED_SETAFFINITY and SYS_SCHED_GETAFFINITY were added in Plan 01.

1. Create `src/user/test_runner/tests/syscall/process_control.zig` with 10 tests:

   Import `syscall` (the `@import("syscall")` pattern used by all test files).

   **Test 1: testPrctlSetGetName**
   - Set name to "mythread" via `syscall.prctl(syscall.PR_SET_NAME, @intFromPtr("mythread"), 0, 0, 0)`
   - Get name into 16-byte buffer via `syscall.prctl(syscall.PR_GET_NAME, @intFromPtr(&buf), 0, 0, 0)`
   - Verify buf starts with "mythread\x00" (compare first 8 bytes + null)
   - Return error.TestFailed if mismatch

   **Test 2: testPrctlGetNameDefault**
   - Get current thread name without setting it first
   - Just verify the syscall succeeds (returns 0, no EFAULT/EINVAL)
   - The default name may be empty or "init" -- don't assert specific content

   **Test 3: testPrctlSetNameTruncation**
   - Set name to "abcdefghijklmnopqrstuvwxyz" (26 chars, exceeds 15-char limit)
   - Get name back
   - Verify name is truncated to 15 chars: first 15 chars match "abcdefghijklmno"
   - Verify null termination at position 15

   **Test 4: testPrctlInvalidOption**
   - Call prctl with option = 9999 (unsupported)
   - Expect error.InvalidArgument (EINVAL)
   - If it succeeds, return error.TestFailed

   **Test 5: testPrctlSetNameEmpty**
   - Set name to "" (empty string, just a null byte)
   - Get name back
   - Verify first byte is 0 (empty name)

   **Test 6: testSchedGetaffinityBasic**
   - Allocate 128-byte mask buffer, zero-init
   - Call `syscall.sched_getaffinity(0, 128, &mask)` (pid=0 = current process)
   - Verify mask[0] has bit 0 set (CPU 0)
   - Verify return value >= 8 (at least 8 bytes written)

   **Test 7: testSchedSetaffinityBasic**
   - Create 8-byte mask with value 1 (CPU 0 only)
   - Call `syscall.sched_setaffinity(0, 8, &mask)` (pid=0)
   - Verify success (no error)

   **Test 8: testSchedSetaffinityMultiCpu**
   - Create 8-byte mask with value 0xFF (CPUs 0-7)
   - Call sched_setaffinity -- should succeed because CPU 0 is included
   - On single-CPU kernel, extra bits are ignored (Linux intersection semantics)

   **Test 9: testSchedSetaffinityNoCpu0**
   - Create 8-byte mask with value 0xFE (CPUs 1-7, NOT CPU 0)
   - Call sched_setaffinity -- should fail with EINVAL (no valid CPUs on single-CPU kernel)

   **Test 10: testSchedGetaffinitySizeTooSmall**
   - Call sched_getaffinity with cpusetsize = 4 (less than minimum 8)
   - Should fail with EINVAL

   For error checking pattern, follow existing test files:
   ```zig
   const result = syscall.prctl(9999, 0, 0, 0, 0);
   if (result) |_| {
       return error.TestFailed; // Should have failed
   } else |err| {
       if (err != error.InvalidArgument) return error.TestFailed;
   }
   ```

   For sched_setaffinity/getaffinity, the mask pointer parameter is `[*]u8`. Pass as `@ptrCast(&mask_buf)` where mask_buf is an appropriately sized array.
