<task type="auto">
  <name>Task 2: Implement memory management no-op syscalls</name>
  <files>src/kernel/sys/syscall/memory/memory.zig</files>
  <action>
    Add 6 new syscall handler functions to the EXISTING memory.zig file (append after sys_brk). These are all no-ops because the kernel does not swap -- pages are always resident and always "locked."

    **sys_madvise(addr: usize, length: usize, advice: usize) SyscallError!usize:**
    - Validate addr is page-aligned: `if (addr & (pmm.PAGE_SIZE - 1) != 0) return error.EINVAL;`
    - Validate length > 0: `if (length == 0) return error.EINVAL;`
    - Validate advice is recognized (0=MADV_NORMAL, 1=MADV_RANDOM, 2=MADV_SEQUENTIAL, 3=MADV_WILLNEED, 4=MADV_DONTNEED, 8=MADV_FREE, 9=MADV_REMOVE, 12=MADV_HUGEPAGE, 13=MADV_NOHUGEPAGE): Accept known values, return EINVAL for unknown. Use a switch statement.
    - Return 0 (no-op).

    **sys_mlock(addr: usize, length: usize) SyscallError!usize:**
    - Validate addr page-aligned. Validate length > 0.
    - Return 0 (no-op -- pages always locked).

    **sys_munlock(addr: usize, length: usize) SyscallError!usize:**
    - Same validation as mlock.
    - Return 0 (no-op).

    **sys_mlockall(flags: usize) SyscallError!usize:**
    - Validate flags: only MCL_CURRENT (1) and MCL_FUTURE (2) bits allowed.
    - `if (flags == 0) return error.EINVAL;`
    - `if (flags & ~@as(usize, 3) != 0) return error.EINVAL;`
    - Return 0 (no-op).

    **sys_munlockall() SyscallError!usize:**
    - Return 0 (no-op, no arguments to validate).

    **sys_mincore(addr: usize, length: usize, vec_ptr: usize) SyscallError!usize:**
    - Validate addr page-aligned. Validate length > 0.
    - Calculate num_pages: `const num_pages = (length + pmm.PAGE_SIZE - 1) / pmm.PAGE_SIZE;`
    - Validate vec_ptr with isValidUserAccess for num_pages bytes (Write mode).
    - Write 1 (resident) to each byte in vec using UserPtr byte-at-a-time: loop `i` from 0 to num_pages, `UserPtr.from(vec_ptr + i).writeValue(@as(u8, 1))`.
    - Return 0.

    Import `pmm` at the top if not already imported (it is -- check). Use the existing `SyscallError`, `UserPtr`, `base` imports already in the file.

    IMPORTANT: The dispatch table in table.zig auto-discovers handlers. The `memory` module is already in the search list (line 88: `else if (@hasDecl(memory, name))`). Since the memory module's root is memory.zig itself (it's not behind a root.zig indirection), the new sys_* functions in memory.zig will be auto-discovered. Verify the build.zig module graph to confirm `memory` maps to `memory/memory.zig`.
  </action>
