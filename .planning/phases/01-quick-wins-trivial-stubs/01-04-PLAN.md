    The wrapper pattern (from existing code like resource.zig `getrandom`):
    ```zig
    pub fn wrapper_name(args...) SyscallError!ReturnType {
        const ret = primitive.syscallN(syscalls.SYS_NAME, @intFromPtr(ptr_arg), scalar_arg, ...);
        if (primitive.isError(ret)) return primitive.errorFromReturn(ret);
        return ret; // or void for 0-returning syscalls
    }
    ```
