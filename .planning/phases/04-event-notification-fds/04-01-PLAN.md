  <name>Task 2: Implement eventfd kernel syscalls with FileOps</name>

  <action>
Create src/kernel/sys/syscall/io/eventfd.zig following the epoll_create1 pattern from scheduling.zig and blocking I/O pattern from pipe.zig:

**EventFdState struct:**
- counter: std.atomic.Value(u64) -- the event counter
- semaphore_mode: bool -- EFD_SEMAPHORE flag
- lock: sync.Spinlock -- protects counter + waiter lists
- blocked_readers: ?*sched.Thread -- thread waiting for counter > 0
- blocked_writers: ?*sched.Thread -- thread waiting for counter < MAX
- reader_woken: std.atomic.Value(bool) -- SMP wakeup flag (pipe.zig pattern)
- writer_woken: std.atomic.Value(bool) -- SMP wakeup flag
- MAX_COUNTER: u64 = 0xfffffffffffffffe

**eventfdRead (FileOps.read):**
- Return EINVAL if buf.len < 8
- Acquire lock, load counter
- If counter == 0: if O_NONBLOCK return EAGAIN, else block using pipe.zig blocking pattern (set blocked_readers to current thread, release lock, check reader_woken flag, call sched.block(), re-acquire lock, re-check counter)
- If semaphore_mode: result = 1, fetchSub(1). Else: result = counter, store(0)
- Wake blocked_writers if any (set writer_woken flag, call sched.unblock)
- Copy u64 result to buf[0..8] using @memcpy + std.mem.asBytes
- Return 8

**eventfdWrite (FileOps.write):**
- Return EINVAL if buf.len < 8
- Read u64 value from buf[0..8]
- Return EINVAL if value == 0xffffffffffffffff (maxInt(u64))
- Acquire lock, load counter
- Check overflow: if counter + value > MAX_COUNTER or wraps: if O_NONBLOCK return EAGAIN, else block using writer blocking pattern
- Store counter + value
- Wake blocked_readers if any
- Return 8

**eventfdPoll (FileOps.poll):**
- Load counter atomically (no lock needed for poll)
- If requested EPOLLIN and counter > 0: set EPOLLIN
- If requested EPOLLOUT and counter < MAX_COUNTER: set EPOLLOUT
- Return revents

**eventfdClose (FileOps.close):**
- Extract EventFdState from private_data, destroy it with heap.allocator().destroy
- Return 0

**eventfd_file_ops constant:**
- read = eventfdRead, write = eventfdWrite, close = eventfdClose, poll = eventfdPoll
- All other ops null (seek, stat, ioctl, mmap, truncate, getdents, chown)

**sys_eventfd2(initval: usize, flags: usize) SyscallError!usize:**
- Validate flags: only EFD_CLOEXEC | EFD_NONBLOCK | EFD_SEMAPHORE allowed, return EINVAL otherwise
- Allocate EventFdState with heap.allocator().create, errdefer destroy
- Initialize with initval (cast to u64) and semaphore_mode from flags
- Allocate FileDescriptor with heap.allocator().create, errdefer destroy
- Set fd.ops = &eventfd_file_ops, fd.flags = O_RDWR | (O_NONBLOCK if EFD_NONBLOCK), fd.private_data = state, fd.refcount = .{ .raw = 1 }, fd.lock = .{}, fd.cloexec = (flags & EFD_CLOEXEC) != 0
- Install via base.getGlobalFdTable().allocAndInstall(fd), return EMFILE on failure
- Return fd_num

**sys_eventfd(initval: usize) SyscallError!usize:**
- Delegate to sys_eventfd2(initval, 0)

**Imports needed:**
- std, heap, fd (the fd module), sched, sync, uapi, base (from ../base.zig or similar -- check how other io syscall files import it)
- Check how read_write.zig and other files in this directory import their dependencies. Follow the same pattern exactly.
