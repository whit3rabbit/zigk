  <action>
Create src/user/test_runner/tests/syscall/event_fds.zig with test functions following the io_mux.zig pattern (pub fn testXxx() !void, using syscall.* wrappers, returning error.TestFailed on assertion failures):

**eventfd tests (EVT-01, EVT-02):**

1. **testEventfdCreateAndClose:** Create eventfd2(0, 0), verify fd >= 0, close it.

2. **testEventfdWriteAndRead:** Create eventfd2(0, EFD_NONBLOCK). Write u64 value 5 to it (construct 8-byte buffer with value 5, call syscall.write). Read 8 bytes. Verify read value == 5 (parse u64 from buffer). Read again -- should return EAGAIN (counter is now 0). Close fd.

3. **testEventfdSemaphoreMode:** Create eventfd2(0, EFD_SEMAPHORE | EFD_NONBLOCK). Write value 3 (counter becomes 3). Read -- should return 1 (semaphore mode). Read -- should return 1 (counter now 1). Read -- should return 1 (counter now 0). Read -- should return EAGAIN (counter is 0). Close fd.

4. **testEventfdInitialValue:** Create eventfd2(42, EFD_NONBLOCK). Read immediately -- should return 42. Close fd.

5. **testEventfdEpollIntegration (EVT-07):** Create eventfd2(0, EFD_NONBLOCK). Create epoll instance. Add eventfd to epoll with EPOLLIN. epoll_wait with timeout=0 -- should return 0 (counter is 0, no EPOLLIN). Write value 1 to eventfd. epoll_wait with timeout=0 -- should return 1 with EPOLLIN. Read from eventfd (consume). epoll_wait with timeout=0 -- should return 0 again. Close both fds.

**timerfd tests (EVT-03, EVT-04, EVT-05):**

6. **testTimerfdCreateAndClose:** Create timerfd_create(CLOCK_MONOTONIC, 0), verify fd >= 0, close it.

7. **testTimerfdSetAndGetTime:** Create timerfd_create(CLOCK_MONOTONIC, TFD_NONBLOCK). Call timerfd_settime with it_value = {0, 100_000_000} (100ms relative), it_interval = {0, 0} (one-shot). Immediately call timerfd_gettime -- it_value should be non-zero (time remaining). Close fd.

8. **testTimerfdExpiration:** Create timerfd_create(CLOCK_MONOTONIC, TFD_NONBLOCK). Set timer for 10ms relative ({0, 10_000_000}). Sleep 50ms using nanosleep (giving timer time to expire). Read from timerfd -- should return expiration count >= 1 (u64). Close fd.

NOTE on sleep: Use syscall.nanosleep or a yield loop to wait. The timer expiration is checked on read via polling the clock, so sleeping gives time to pass.

9. **testTimerfdDisarm:** Create timerfd_create(CLOCK_MONOTONIC, TFD_NONBLOCK). Set timer for 100ms. Immediately disarm by setting it_value to {0, 0}. Sleep 150ms. Read -- should return EAGAIN (timer was disarmed before expiry). Close fd.

**signalfd tests (EVT-06):**

10. **testSignalfdCreateAndClose:** Create signalfd4(-1, &mask, 0) with mask containing SIGUSR1 (signal 10, mask = 1 << 9). Verify fd >= 0. Close it.

11. **testSignalfdReadSignal:** Create signalfd4(-1, &mask, SFD_NONBLOCK) with SIGUSR1 in mask. Send SIGUSR1 to self via kill(getpid(), SIGUSR1). Read 128 bytes from signalfd. Verify ssi_signo == 10 (SIGUSR1). Read again -- should return EAGAIN (signal consumed). Close fd.

NOTE: Before sending the signal, block SIGUSR1 with rt_sigprocmask to prevent the default signal handler from running. The signal will be pending and readable from signalfd.

12. **testSignalfdEpollIntegration (EVT-07):** Create signalfd4(-1, &mask, SFD_NONBLOCK) with SIGUSR1. Create epoll instance, add signalfd with EPOLLIN. epoll_wait timeout=0 -- should return 0. Block SIGUSR1 and send it to self. epoll_wait timeout=0 -- should return 1 with EPOLLIN. Read from signalfd (consume). epoll_wait timeout=0 -- should return 0. Close fds.

**Implementation notes:**
- Use `const syscall = @import("syscall");` (same as other test files)
- For writing u64 to eventfd: `var val: u64 = X; _ = try syscall.write(fd, @as([*]const u8, @ptrCast(&val)), 8);`
- For reading u64: `var val: u64 = 0; _ = try syscall.read(fd, @as([*]u8, @ptrCast(&val)), 8);`
- For signalfd reading: allocate 128-byte buffer, cast first 4 bytes to u32 for ssi_signo
- Check what signal-related wrappers exist in syscall lib (kill, rt_sigprocmask, getpid)
- If any test is difficult due to missing infrastructure (e.g., rt_sigprocmask not available in userspace), skip it and document why
