Create directory `src/net/transport/tcp/congestion/` and file `reno.zig`.

The module exports three `pub fn` entry points that mutate `*Tcb` in-place. All callers already hold tcb.mutex. No heap allocation.

**`pub fn onAck(tcb: *Tcb, acked_bytes: u32) void`**
Handles all cases: fast recovery (full ACK vs partial ACK) and normal (slow-start vs congestion avoidance).

Fast recovery path:
- If `tcb.fast_recovery` is true:
  - Full ACK (seqGte(tcb.snd_una, tcb.recover)): exit fast recovery, set cwnd = ssthresh
  - Partial ACK: set cwnd = ssthresh + mss (caller handles retransmit BEFORE calling onAck)
  - Apply capCwnd and return

Normal path:
- Slow-start (cwnd < ssthresh): cwnd += min(acked_bytes, mss) using std.math.add with maxInt(u32) catch
- Congestion avoidance (cwnd >= ssthresh): cwnd += max(1, (mss * mss) / cwnd) -- compute in u64 to avoid overflow, truncate to u32
- Apply capCwnd

**`pub fn onTimeout(tcb: *Tcb) void`**
RFC 5681 S3.5 + RFC 6298:
- ssthresh = max(flight_size / 2, 2 * mss) where flight_size = snd_nxt -% snd_una
- cwnd = mss (1*SMSS, NOT IW10)
- fast_recovery = false
- rtt_seq = 0 (Karn's Algorithm, CC-03)

**`pub fn onDupAck(tcb: *Tcb, dup_count: u8) void`**
RFC 5681 S3.2:
- If dup_count == 3 and NOT already in fast_recovery:
  - ssthresh = max(flight / 2, 2 * mss)
  - cwnd = ssthresh + 3 * mss
  - fast_recovery = true
  - recover = snd_nxt
- Else if already in fast_recovery:
  - cwnd += mss (inflate) using std.math.add with maxInt(u32) catch
  - Apply capCwnd

**`fn capCwnd(tcb: *const Tcb) u32`** (private)
Returns @min(tcb.cwnd, constants.MAX_CWND). This enforces CC-05.

Imports:
```zig

const seqGte = types.seqGte;
```

IMPORTANT: The `mss` field on Tcb is `u16`. When used in arithmetic with u32 values, cast explicitly: `@as(u32, tcb.mss)`.
